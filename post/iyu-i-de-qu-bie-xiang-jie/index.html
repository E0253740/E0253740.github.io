i++与++i的区别详解

2021-03-29

周六做京东笔试的时候遇到了对于i++与++i的考察，正好今天想起来，就仔细深入的研究了一下
先来看几个例子：

Sample 1
int i = 0;
i = i++;
System.out.println("i = " + i);
正确答案：0
详解：
先看i++，根据原理“先自增，然后返回自增之前的值”，i 自增后，i = 1，但是接着返回自增之前的值0，此时表达式变成 i = 0，0没有赋值给 i 时 i 的值是1，但是当把0赋值给 i 时，i 的值就又变成0了。因此 i = i++ 这句代码是做无用功，因为 i 的值最终还是和原来一样。

Sample 2
int a = 2;
int b = (3 * a++) + a;
System.out.println(b);
正确答案：9

Sample 3
int a = 2;
int b = a + (3 * a++);
System.out.println(b);
正确答案：8

Sample 4
int i = 1;
int j = 1;
int k = i++ + ++i + ++j + j++;
System.out.println(k);
正确答案：8

Sample 5
int a = 0;
int b = 0;
a = a++;
b = a++;
System.out.println("a = " + a + ", b = " + b);
正确答案：a=1 b=0

基本原理是：
i++ 即后加加，原理是：先自增，然后返回自增之前的值
++i 即前加加，原理是：先自增，然后返回自增之后的值

i++的基本原理可以用下面这个代码表示
int temp = i;
i = i + 1;
return temp;
